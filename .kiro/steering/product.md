# Flappy Kiro — ภาพรวมผลิตภัณฑ์

Flappy Kiro เป็นเกม endless scroller สไตล์ retro ที่เล่นบนเว็บเบราว์เซอร์ผ่าน HTML5 Canvas ผู้เล่นควบคุมตัวละคร "Ghosty" ให้บินหลบท่อที่เลื่อนเข้ามาจากด้านขวา โดยกดเพื่อกระพือปีก (คลิกเมาส์ / แตะหน้าจอ / กด Space)

## ฟีเจอร์หลัก

- ระบบฟิสิกส์แรงโน้มถ่วงและการกระพือปีก (gravity + flap)
- ท่อสีเขียวสุ่มตำแหน่งช่องว่าง เลื่อนจากขวาไปซ้าย
- ระบบคะแนนพร้อม high score บันทึกผ่าน localStorage
- เมฆพื้นหลังเคลื่อนไหวแบบ parallax (opacity สัมพันธ์กับความเร็ว)
- ระบบเสียง (กระโดด / Game Over) พร้อม fallback เมื่อโหลดไม่ได้
- Impossible Mode — Red Gate (ท่อสีแดง) ที่กลับทิศแรงโน้มถ่วงชั่วคราว 10 วินาที พร้อม countdown timer

## สถานะเกม (State Machine)

- `IDLE` → แสดงหน้าจอเริ่มต้น รอผู้เล่นกด
- `PLAYING` → เกมกำลังดำเนินอยู่
- `GAME_OVER` → แสดงคะแนน รอกดเพื่อเริ่มใหม่

## ตัวละครและ Entity

- **Ghosty** — ตัวละครผี ควบคุมด้วย flap, มี bounding box 40x40
- **Pipe (ท่อเขียว)** — ท่อบน-ล่างมีช่องว่างตรงกลาง ผ่านได้ +1 คะแนน
- **Red Gate (ท่อแดง)** — เหมือน Pipe แต่ผ่านแล้วกลับทิศแรงโน้มถ่วง
- **Cloud (เมฆ)** — decoration พื้นหลัง เคลื่อนที่ช้ากว่าท่อ
